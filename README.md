🎓 College Project Collaboration Platform (Frontend)

A modern and responsive React frontend application that helps students discover, create, and collaborate on college projects across domains like Web Development, Mobile Apps, IoT, and Research.

🚀 Features
🔍 Project Discovery – Browse and search projects by category or keywords
👤 User Dashboard – View your projects, collaborations, and activity
➕ Project Creation – Post new project ideas and find teammates
🔐 Authentication UI – Login & Register pages (frontend only)
📱 Responsive Design – Works on desktop, tablet, and mobile
🛠 Tech Stack
React.js
CSS3 (Flexbox, Media Queries, Variables)
React Router DOM
Axios

⚙️ Getting Started
Prerequisites
Node.js installed
Installation
git clone https://github.com/your-username/project-collab-platform.git
cd frontend
npm install
Run the App
npm start

Open: http://localhost:3000

🔗 Routes
/ → Projects
/login → Login Page
/register → Register Page
/dashboard → Dashboard
/projects → Browse Projects
/create → Create Project
